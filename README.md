# Kanban Board

## Description

- Kanban board with a secure login page to access the board of tasks.

## Installation

- npm install
- psql -U postgress; (create data base and seed database)
- npm run build
- npm start:dev

- Link to deployed website: https://kanban-board-ewu5.onrender.com
